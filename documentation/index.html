esve: a library for 3D and 4D visualization

Included applications (apps/ directory):
  • hopfviewer: the four-dimensional Clifford torus. The in-application help is supplemented by this readme.
  • topeviewer: 4D polytopes. Some sample data files are included. The in-application help is supplemented by this readme.
  • polynomialviewer: drawing real curves of polynomials. See the in-application help.
Example mini-applications (examples/ directory):
  • waves, monomial, and poles are examples of user-defined parametric surfaces in 3D and 4D. For details see the readme for surfaces.
  • simple3d and simple4d are tutorials for Simple_Viewer, a simplified front-end to the esve library. See the tutorial readme.
  • A handful of rudimentary tutorials in examples/basic/.
